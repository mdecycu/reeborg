Development

目前 https://mde.tw/reeborg 使用 Brython 3.2.4, 希望能修改為最新版的 Brython.

另外, 程式中使用的許多 Javascript 程式庫, 也希望改為較新的版本.

最後, Reeborg 中的機器人系統, 將利用 CoppeliaSim (教育單位免費使用) 與 Webots (採 Apache 2.0 授權) 同步使用 Python 程式控制 3D 版的機器人系統, 並朝著 Cyber-Physical System (虛實整合系統) 的應用發展.

用來開發 3D 零組件系統的 CAD 將採用 Solvespace 以及 Siemens NX2312.

系統開發採用 Leo Editor.
